# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.x     | ✅        |

## Reporting a Vulnerability

Please report security vulnerabilities by opening a **private** GitHub Security Advisory at:
https://github.com/exisz/dr-agent/security/advisories/new

Do **not** open a public issue for security vulnerabilities.

We aim to respond within 48 hours and release a patch within 7 days for confirmed issues.
